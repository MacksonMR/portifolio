Bem vindo ao Angular

Esse projeto mostra de forma muito simples a integração do angular com C# por meio de API. Onde são executados dois projetos.

  • Projeto em C#, formando uma API RestFull com os metodos:
    • Create: Criar uma nova "familia".
    • Delete: Deletar uma "familia" já existente que tenha o id igual ao da requisição.
    • Update: Atualizr o registro de uma "familia" já existente.
    • Getall: Retornar todas as "familias" existente.
    • GetById: Retorna a "familia" que tenha o mesmo id da requisição.
  • Projeto em Angular, realizando requisições para API conforme é necessitado pelo usuario

Geralmente as ações no angular, não geram "Reload" da pagina e sim a "Renderização" do(s) componente(es), isso gera o aumento na performance.
